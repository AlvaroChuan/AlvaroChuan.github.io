Wave Function Collapse Tool for Unity

Tool for the procedural generation of 3D environments using a GPU-parallelized version of the Wave Function Collapse algorithm, integrated within the Unity engine. The tool enables users to define custom tile sets and adjacency rules to generate worlds, offering an intuitive interface to manage tile and world creation. This tool, along with the game below are the result of my Bachelor's thesis done in 2025, that where awarded the hightest grade in the subject and honors.

Interactive Bézier-Based 2D Extrusion Tool For Unity

Evolution of the tool used in Project: Rush! This is a tool for Unity that allows the user to generate procedural meshes based on Bezier curves inside Unity's editor. The user can provide any 2D shape as the base of the mesh and then the tool will extrude it along the Bezier curve generating a 3D mesh. The user can also modify the shape of the mesh by modifying the control points of the curve, twisting, rotating or moving the mesh. The tool also allows the use to change the ammount of segments of the mesh and store this mesh to use it in the game. A short paper was published in the CEIG 2025 conference about the tool and its benefits for the game development process.
